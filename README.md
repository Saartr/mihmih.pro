# Дизайн-система Скворцова

Дизайн-система для портфолио **Михаила Скворцова** — руководителя направления проектирования интерфейсов в НОТА (Группа компаний Т1). Сайт-визитка для технических рекрутёров и hiring-менеджеров: главная, проекты, резюме, контакты.

> Минималистичная швейцарская типографика. Чёрный, белый, серый — плюс одна акцентная сигнально-оранжевая. Inter + JetBrains Mono. Жёсткая сетка, разреженный ритм, никакой декоративности.

## Основы контента

**Язык.** Только русский. Английский допустим в технических терминах (B2B, ATS, Storybook, NPS). Никаких английских вкраплений в тоне.

**Тон.** Спокойный, профессиональный, без саморекламы. Минимум прилагательных. Числа важнее эмоций. Фразы короткие, утвердительные. Не «помогаю компаниям расти» — а «15+ лет в продуктовом дизайне».

**Регистр.** Sentence case в заголовках. ВЕРХНИЙ РЕГИСТР только в моноширинных eyebrow-метках с разрядкой (`СЕКЦИЯ`, `01 / ИНДЕКС`).

**Я vs мы.** От первого лица единственного числа: «проектирую», «руковожу», «веду». Никакого «мы» — это личное портфолио.

**Длина.** Один большой смысл на экран. На главной — 6–12 слов в герое. Описание проекта — два-три предложения максимум до клика.

**Эмодзи.** Не используются. Совсем.

**Числа.** Табулярные, моноширинные. `+15 лет`, `+42% к скорости задач`, `NPS +68`. Знак перед числом.

**Примеры копирайта**

- Hero: *«Проектирую интерфейсы B2B и B2C продуктов. 15+ лет.»*
- Подзаголовок: *«Руководитель отдела проектирования в НОТА (Группа Т1). Дизайн-системы, корпоративные продукты, финтех.»*
- Eyebrow: `01 — ВЫБРАННЫЕ РАБОТЫ`
- CTA: «Связаться» (не «Давайте поработаем!»)
- Подпись проекта: «НОТА Юнион · HRM-платформа · 2022—2023»

## Визуальные основы

**Палитра.** Бумажный фон `#FAFAF7`. Текст `#0A0A0A`. Серая шкала из пяти ступеней (`fg-2..fg-5`). Один акцент — сигнально-оранжевый `#FF4500` — используется штучно: на одной кнопке экрана, на маркере «активно», на 1–2 словах в заголовке. Никаких градиентов. Тёмная поверхность `#0E0E0E` существует как контрастная full-bleed секция (резюме, футер).

**Типографика.** Inter (sans, основной) + JetBrains Mono (моно — для метаданных, eyebrow-меток, чисел, технических подписей). Sans для всего читаемого; Mono для всего, что измеряет, маркирует или индексирует. Display 72–128px со срезанным трекингом (-0.04em). Body 15px. Mono 11–13px с разрядкой 0.12em для секций.

**Сетка.** 12 колонок, гаттер 24px, контейнер 1280px (узкий 960px для текста). Жёсткое выравнивание по бэйслайну. Колонок намного больше, чем нужно — большинство элементов занимает 4–6 колонок и оставляет тонну пустоты.

**Фон.** Никаких изображений-подложек, паттернов, текстур, иллюстраций. Бумажный однотонный фон. Изображения — только полноразмерные продуктовые скриншоты в рамках браузеров и устройств. Полноэкранные мобильные мокапы на нейтральном фоне.

**Анимация.** Минимум. Subtle fade + slide-up (12–16px) при появлении секции, длительность 280ms, easing `cubic-bezier(0.22, 1, 0.36, 1)` (out-expo). Никаких bounce, spring, parallax, magnet-курсоров. Курсор меняется на pointer — этого достаточно.

**Hover.** Текстовые ссылки — подчёркивание появляется через `text-decoration: underline` с `text-underline-offset: 4px`. Кнопки — затемнение (чёрная → `#1A1A1A`, оранжевая → `#E63E00`). Карточки проектов — лёгкий сдвиг (translateY -4px) и появление мета-стрелки `→`.

**Press / active.** Без shrink. Просто следующий шаг затемнения. На клавиатуре — outline `2px solid var(--accent)` с offset 2px.

**Границы.** Hairline `1px solid var(--line)` (#E5E4DF) — для разделителей строк, таблиц, карточек. Heavy rule `1px solid var(--line-strong)` (#1A1A1A) — для семантических разделителей секций. Никаких dashed/dotted.

**Тени.** Почти не используются. Если нужно — `--shadow-1` (1px hairline-тень для отделения карточки от фона), `--shadow-2` (для одного floating-элемента на странице, например модалки). По умолчанию — flat.

**Прозрачность и blur.** Только в одном месте: sticky header при скролле получает `backdrop-filter: blur(12px)` + `background: rgba(250, 250, 247, 0.8)`. Больше нигде.

**Углы.** Острые. `--r-0` (0px) для большинства поверхностей. `--r-2` (4px) для кнопок и инпутов. `--r-pill` для тегов-капсул в мета-блоках. Никаких больших скруглений.

**Карточки.** Без скруглений, без тени, без рамки по умолчанию. Карточка проекта = полноширинное изображение + мета-строка ниже. Разделение между карточками — пустота, не линия.

**Цвет в изображениях.** Скриншоты — нейтральные, светлые интерфейсы (как и сама система). Если есть тёмная UI — это исключение. Без warm grain, без duotone, без фильтров.

**Правила раскладки.** Sticky header (тонкий, 56px). Логотип слева, навигация справа. Footer — full-bleed тёмная секция с контактами и моно-метаданными. Большие блоки разделены `--sp-10` (128px) на десктопе.

## Иконография

**Подход.** Иконки — последняя мера. По возможности заменять моноширинной меткой или числом. Когда необходимо — тонкие линейные SVG, толщина обводки 1.5px, размер 16/20/24, без заливки, без скруглений на концах (кроме `stroke-linecap: round` для линий, нарисованных вручную).

**Источник.** [Lucide](https://lucide.dev) (CDN: `https://unpkg.com/lucide@latest/dist/umd/lucide.min.js`). Тонкая, нейтральная, монолинейная — совпадает с типографической логикой. Альтернатива — Phosphor Thin, если захочется чуть тоньше.

> Подстановка: оригинальных иконок у бренда нет — выбран Lucide как наиболее нейтральный CDN-набор. Это можно заменить, если появятся собственные.

**Используемые иконки**: `arrow-up-right` (внешние ссылки), `arrow-right` (читать дальше), `external-link`, `mail`, `phone`, `send` (Telegram), `chevron-down`, `x` (закрыть). Список ограничен — больше не вводить без необходимости.

**Эмодзи.** Не используются.

**Юникод.** Допустим как типографический инструмент, не как иконки: `→` (стрелка-связка), `·` (разделитель в мета-блоках), `—` (длинное тире для диапазонов лет), `№`. Все из основного font-stack.

**Логотип.** Монограмма «МС» с акцентной точкой. Чёрная на светлом, светлая на тёмной поверхности. Минимальный размер — 24px по высоте (тогда точка превращается в 1.5px).

## Открытые вопросы

- Шрифты Inter и JetBrains Mono подгружаются с Google Fonts — у бренда нет собственной кастомной типографики. Если хочется характерных русских кириллических букв — рассмотреть Manrope, Suisse Int'l (платный) или Onest.
- Все продуктовые скриншоты в UI-ките — плейсхолдеры с подписями. Для полноценного портфолио нужны реальные скриншоты НОТА Юнион, nota.tech, diongo.ru, Монополия.
- Логотип-монограмма создан с нуля. Это рабочий вариант — финальный знак стоит делать отдельным заходом.
