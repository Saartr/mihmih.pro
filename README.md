# mihmih.pro

Personal portfolio website for Mikhail Skvortsov.

The project is a static HTML/CSS/JavaScript site without a build step. It contains portfolio sections, case studies, custom cursor behavior, image loading states, lightbox previews, sticky navigation, and project-side metadata blocks.

## Structure

```text
.
|-- index.html      # Page markup and content
|-- styles.css      # Layout, responsive styles, animations
|-- portfolio.js    # Interactions, cursor, lightbox, sticky elements
`-- assets/         # Images, SVGs, icons, project media
```

## Local Development

Open `index.html` directly in a browser, or run a small local server from the project root:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```
