# Portfolio UI Kit — Скворцов

Кликабельный прототип сайта-портфолио. Главная, работы, резюме, контакты.

## Файлы

```
index.html         — точка входа, маршрутизация между секциями
Header.jsx         — sticky-шапка с навигацией
Footer.jsx         — тёмный футер с контактами
Hero.jsx           — главная (display + краткое описание + 2 CTA)
WorkIndex.jsx      — индекс проектов (таблица-список)
CaseStudy.jsx      — отдельный кейс (full-bleed shot + meta + текст)
Cv.jsx             — резюме (timeline + skills)
Contact.jsx        — форма + прямые контакты
ProjectCard.jsx    — карточка проекта (плейсхолдер-скриншот + мета)
Button.jsx · Tag.jsx · Components.jsx — общие примитивы
```

## Запуск

Открыть `index.html`. Прототип переключает секции через клиентский роутер на `useState`.

## Caveats

- Скриншоты проектов — плейсхолдеры (CSS-моки браузерных окон). Для реального сайта нужны настоящие изображения НОТА Юнион / nota.tech / Монополия.
- Форма «Связаться» не отправляет — только UI.
